FUNCTIONS

1. Function (or) Mapping :

Let $X$ and $Y$ be any two non-empty sets and there be correspondence or association between the elements of $X$ and $Y$ such that for every element $x \in X$, there exists a unique element $y \in Y$, written as $y = f(x)$. Then we say that $f$ is a mapping or function from $X$ to $Y$, and is written as $f : X \to Y$ such that $y = f(x), x \in X, y \in Y$.

Note The number of functions that can be defined from $A$ to $B$ is $[n(B)]^{n(A)}$.
2. Domain and range of a function :

If $f : X \to Y$ be a function, then the set $X$ is said to be the domain of $f$ and range of $f$ is set of all image points in $Y$ under the map $f$.
i.e., $f(X) = \{f(x) : f(x) \in Y; x \in X\}$.
The set $Y$ is also called the co-domain of $f$. Clearly $f(X) \subseteq Y$.

3. Equality of functions :

Two functions $f$ and $g$ are said to be equal iff,
i) the domain of $f$ = the domain of $g$
ii) $f(x) = g(x) \ \forall x \in \text{domain}$.

4. Logarithmic Function :

$\log_b a$ is defined when $a > 0, b > 0$ and $b \neq 1$.

5. Exponential Functions :

$a^x$ is defined for all real values of $x$, where $a > 0$.

6. Modulus function (or) Absolute value function :

The function $f : R \to R$, is defined as $$f(x) = |x| = \begin{cases} x, & \text{if } x > 0 \\ 0, & \text{if } x = 0 \\ -x, & \text{if } x < 0 \end{cases}$$ is called the absolute value function or modulus function. Its domain is $R$ and its range is $[0, \infty)$.

7. Greatest integer (Step or integral) function :

The function $f : R \to R$, is defined as $f(x) = [x]$ is called the greatest integer function, where $[x]$ = integral part of $x$ or greatest integer not greater than $x$ or greatest integer less than or equal to $x$.
i.e. $f(x) = n$, where $n \leq x < n+1, n \in I$ (the set of integers).
Its domain is $R$ and range is $I$.

Note i) $[x] \leq x < [x]+1$
ii) $[x+y]=[x]+[y]$ if either $x$ (or) $y$ (or) both integers
iii) $n \leq x < n+1 \iff [x]=n$
iv) $x-1 < [x] \leq x$
v) $[[x]] = [x]$
vi) $[x]+[-x] = \begin{cases} 0 & \text{if } x \in I \\ -1 & \text{if } x \notin I \end{cases}$
8. Fractional-part function :

The function $f : R \to R$ is defined as $f(x) = x - [x]$ (or) $f(x) = \{x\}$, where $\{x\}$ denotes the fractional part of $x$, is called the fractional part function. Its domain is $R$ and range is $[0, 1)$.

Note i) If $x$ is an integer, then $x = [x] \implies \{x\} = 0 \implies [\{x\}] = 0$
ii) $[\{x\}] = 0$
iii) $\{x\} + \{-x\} = \begin{cases} 0, & \text{if } x \in \text{integer} \\ 1, & \text{if } x \notin \text{integer} \end{cases}$
9. Signum function :

The function $f : R \to R$ defined as, $$f(x) = \begin{cases} \frac{|x|}{x} & \text{for } x \neq 0 \\ 0 & \text{for } x = 0 \end{cases}$$ is called the signum function. Its domain is $R$ and range is the set $\{-1, 0, 1\}$.

10. Composite Function :

Let $f$ and $g$ be two real functions with domain $D_1$ and $D_2$ respectively. If range of $f \subseteq$ domain of $g$, then composite function $(gof)$ is defined by $(gof)(x) = g(f(x)), \forall x \in D_1$.
Also, if range of $g \subseteq$ domain of $f$, then composite function $(fog)$ is defined by $(fog)(x) = f(g(x)), \forall x \in D_2$.

11. Odd and Even functions :

Odd function : A function $f(x)$ is said to be odd if $f(-x) = -f(x)$ for every real number $x$ in the domain of $f$.
Even function : A function $f(x)$ is said to be even if $f(-x) = f(x)$ for every real number $x$ in the domain of $f$.

Note i) The graph of an odd function is symmetric about origin and it is placed either in the first and third quadrant or in the second and fourth quadrant.
ii) The graph of an even function is symmetric about the y-axis.
iii) If a function $f(x)$ is such that $f(a-x) = f(a+x)$ then it is symmetric with respect to the line $x = a$.
iv) If a function $f(x)$ is such that $f(a-x) + f(a+x) = 0$ then it is symmetric with respect to the point $(a, 0)$.

12. Constant Function :

A function $f : R \to R$ defined as $f(x) = c, \forall x \in R$, where $c$ is a constant, is called a constant function. Its domain is $R$ and range is singleton set $\{c\}$. The graph of constant function is a straight line parallel to x-axis when $x$ is the independent variable.

13. Identity function :

The function $f : R \to R$ defined as $f(x) = x, \forall x \in R$, is called the identity function. Its domain is $R$ and range is also $R$. The graph of the identity function is a straight line passing through origin and inclined at an angle of $45^\circ$ with x-axis.

14. Algebraic Functions :

If $f$ and $g$ are real valued functions with domain $A$ and $B$ respectively, then both $f$ and $g$ are defined on $A \cap B$ when $A \cap B \neq \phi$.
i) $(f \pm g)(x) = f(x) \pm g(x)$ and $(fg)(x) = f(x)g(x)$.
ii) Let $f : A \to R$ and $k$ be a constant function defined on $A$ then $(f + k)(x) = f(x) + k$ and $(kf)(x) = kf(x)$ for all $x \in A$.
iii) Let $f : A \to R, g : B \to R, C = \{x \in A \cap B | g(x) \neq 0\} \neq \phi$ then $\frac{f}{g}$ defined on $C$ as $(\frac{f}{g})(x) = \frac{f(x)}{g(x)}$ for all $x \in C$.
iv) Let $f : A \to R$ and $n \in N$ then $|f|, f^n$ are defined on $A$ as $|f|(x) = |f(x)|$ and $f^n(x) = (f(x))^n$ for all $x \in A$.
v) Let $f : A \to R, C = \{x \in A | f(x) \geq 0\} \neq \phi$, then $\sqrt{f}$ defined on $C$ as $\sqrt{f}(x) = \sqrt{f(x)}$ for all $x \in C$.

16. One-one (or) injective function :

A function $f : X \to Y$ is said to be one-one or injective if distinct elements of $X$ have distinct images in $Y$.

17. Many-one function :

A function $f : X \to Y$ is said to be many-one if there exists atleast two distinct elements in $X$ whose images are same.

18. Working Rule for checking whether the function $f : X \to Y$ is one-one (or) many-one :
  • * Consider any two points $x_1, x_2 \in X$.
  • * Put $f(x_1) = f(x_2)$ and solve the equation.
  • * If we get $x_1 = x_2$ only, then $f$ is one-one, otherwise it is many-one.

Horizontal line test :
* If any straight line parallel to x-axis intersects the graph of the function atmost at one point, then the function is one-one, otherwise it is many-one (i.e. it intersects the graph of the function in atleast two points).
* All even functions are many-one.

Note i) If $X$ and $Y$ are any two finite sets having $m$ and $n$ elements respectively, then the number of one-one functions from $X$ to $Y$ would be $= \begin{cases} ^nP_m, & \text{if } n \geq m \\ 0, & \text{if } n < m \end{cases}$.
ii) If $n(A) \leq n(B)$ then the number of many to one functions from $A$ to $B$ is $[n(B)]^{n(A)} - \phantom{.}^{n(B)}P_{n(A)}$.
19. Onto (or) surjective function :

A function $f : X \to Y$ is said to be onto or surjective if every element of $Y$ is the image of some element of $X$ under the map $f$.
A function $f : X \to Y$ is an into function if it is not an onto function.

Note i) If $X$ and $Y$ are any two finite sets having $m$ and $n$ elements respectively, where $1 \leq n \leq m$, then the number of onto functions from $X$ to $Y$ is given by $\sum_{r=1}^n (-1)^{n-r} \phantom{.}^nC_r r^m$ (or) $^nC_0 n^m - \phantom{.}^nC_1(n-1)^m + \phantom{.}^nC_2(n-2)^m - \phantom{.}^nC_3(n-3)^m + \dots$.
ii) The number of into functions from $A$ to $B$ is given by $^nC_1(n-1)^m - \phantom{.}^nC_2(n-2)^m + \phantom{.}^nC_3(n-3)^m - \dots$.
iii) If $n(A) \geq n(B)$ and $n(B) = 2$ then the number of onto functions from $A$ to $B$ are $2^{n(A)} - 2$.
20. Working Rule for checking whether the function $f : X \to Y$ is onto or into :
  • i) Find the range of the function $f$.
  • ii) If range of $f = Y$, then $f$ is onto, otherwise it is into.
21. Bijective function :

A function $f : X \to Y$ is said to be bijective, if $f$ is both one-one and onto.

Note If $X$ and $Y$ are any two finite sets having the same number of elements, say $n$, then the number of bijective functions from $X$ to $Y$ is $n!$.
22. Some useful results :

Let $f : X \to Y$ and $g : Y \to Z$
i) If both $f$ and $g$ are one-one, then $gof$ is also one-one.
ii) If both $f$ and $g$ are onto, then $gof$ is also onto.
iii) If $gof$ is one-one, then $f$ is one-one but $g$ may not be one-one.
iv) If $gof$ is onto, then $g$ is onto but $f$ may not be onto.
v) If $f$ and $g$ are bijective, then $gof$ is also bijective.
vi) It may happen that $gof$ may exist and $fog$ may not exist. Moreover, even if both $gof$ and $fog$ exist, they may not be equal.

23. Inverse Function :

If the function $f : X \to Y$ is both one-one and onto, then we define inverse function $f^{-1} : Y \to X$ by the rule $y = f(x) \iff f^{-1}(y) = x, \forall x \in X, \forall y \in Y$.

24. Working Rule to find the Inverse of a function :

Let $f : X \to Y$ be a bijective function.
* Put $f(x) = y$.
* Solve the equation $y = f(x)$ to obtain $x$ in terms of $y$.
* Interchange $x$ and $y$ to obtain the inverse of $f$.

Note i) The graph of $y = f(x)$ and its inverse written in the form $y = g(x)$ are symmetrical about the line $y = x$.
ii) The graph of $y = f(x)$ and $y = f^{-1}(x)$ if intersects then meet on the line $y = x$ only. Hence, the solution of $f(x) = f^{-1}(x)$ are also the solution of $f(x) = x$.
iii) If $f : A \to B$ and $g : B \to C$ are two bijections, then $gof : A \to C$ is bijection and $(gof)^{-1} = f^{-1} \circ g^{-1}$.
25. Working Rules for finding the domain of a function :

Algebraic Functions :
i) Denominator should be non-zero.
ii) Expression under the even root should be non-negative i.e., $\geq 0$.
Trigonometric Functions :
i) $\sin x$ and $\cos x$ are defined for all real values of $x$.
ii) $\tan x$ and $\sec x$ are defined for all real values of $x$ except $x = (2n+1)\frac{\pi}{2}$, where $n \in I$.
iii) $\cot x$ and $\text{cosec } x$ are defined for all real values of $x$ except $x = n\pi$, where $n \in I$.

Inverse Trigonometric Functions :
i) $\sin^{-1} x$ and $\cos^{-1} x$ are defined for $-1 \leq x \leq 1$.
ii) $\tan^{-1} x$ and $\cot^{-1} x$ are defined for all $x \in R$.
iii) $\sec^{-1} x$ and $\text{cosec}^{-1} x$ are defined for $x \leq -1$ or $x \geq 1$.

26. Rules for solving problems on domain of a function :
  • i) $(x-a)(x-b) > 0 \implies x < a$ or $x > b$, for $a < b$.
  • ii) $(x-a)(x-b) < 0 \implies a < x < b$ for $a < b$.
  • iii) $|x| < a \implies -a < x < a$.
  • iv) $|x| > a \implies x < -a$ or $x > a$.
  • v) $\log_b a > k \implies \begin{cases} a > b^k, \text{ if } b > 1 \\ a < b^k, \text{ if } b < 1 \end{cases}$.
  • vi) $\sqrt{x^2} = |x|$.
  • vii) $\sqrt[n]{x^n} = |x|$, if $n$ is even and $\sqrt[n]{x^n} = x$, if $n$ is odd.
  • viii) Domain of $\frac{1}{\sqrt{f(x)}}$ is $\{x / f(x) > 0\}$.
  • ix) Domain of $\log f(x)$ is $\{x / f(x) > 0\}$.
  • x) Domain of $1/\log f(x)$ is $\{x / f(x) > 0 \text{ and } f(x) \neq 1\}$.
27. Working Rules for finding the range of a function $y = f(x)$ :
  • i) Find the domain of the function $y = f(x)$.
  • ii) If the domain is a set containing finite number of elements then find the corresponding images under the given function and the set of all images so obtained is the range.
  • iii) If the domain is either $R$ or a subset of $R$ then solve the equation $y = f(x)$ and find $x$ in terms of $y$ to get $x = g(y)$, find the values of $y$ for which $y$ is real and defined and the set of values of $y$ so obtained constitutes the range of $f$. Note that if finite number of values of $x$ are excluded from the domain, find the values of $y$ for these values of $x$ and exclude these values of $y$ from the range of $f$ found earlier.
  • iv) In case of monotonic functions find the least and greatest value of $y$ for values of $x$ in the domain. If $a$ is the least value and $b$ the greatest value of $y$ then range $(f) = [a, b]$.
